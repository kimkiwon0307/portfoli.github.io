팀 프로젝트

국비교육 받으면서 했던 프로젝트로 스포츠팀 구하기 사이트를 만들었습니다.

저는 나의 팀 관리, 고객센터를 맡아 개발하였습니다.



사용 기술

Back-End : JAVA, JSP&Servlet, Spring Framework, Mybatis, Tomcat
Front-End : HTML5, CSS, JavaScript, jQuery, BootStrap Framework
Data-Base : MySQL

주요 기능

1. 게시판 CRUD : 쓰고 조회하고 수정하고 삭제하는 기능을 만들었습니다.

2. 글 잠금(자물쇠) 기능 : 작성할 때 설정하여 모두 보기와 작성자만 보기를 구현하였습니다.

3. 댓글 기능 : 고객센터에 질문 글을 남기면 운영자가 댓글로 답변하는 것을 구현하였습니다.
